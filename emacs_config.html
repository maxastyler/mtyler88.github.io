2 basic Setup

2.1 Set Some Default Variables

(setq-default
 confirm-kill-emacs 'yes-or-no-p    ; confirmation when killing emacs
 indent-tabs-mode nil           ; don't use tabs to indent
 select-enable-clipboard t      ; use the system clipboard
 inhibit-startup-screen t
 initial-scratch-message nil
 initial-major-mode 'org-mode
 tab-width 4)               ; tab width to 4

2.2 yes-or-no to y-or-n

Stop asking for me to type in the full "yes" or "no" when a "y" or "n" would do… 

(fset 'yes-or-no-p 'y-or-n-p)

2.3 Turn off cursor blinking

(blink-cursor-mode 0)

2.4 Set a Backup Directory

Use this directory instead of pumping out backup files into the wild 

(setq backup-directory-alist '(("." . "~/.emacs.d/backup"))
      backup-by-copying t
      version-control t
      delete-old-versions t
      kept-new-versions 20
      kept-old-versions 5)

2.5 Add cargo bin to exec path

(add-to-list 'exec-path "~/.cargo/bin")

3 Other Packages

3.1 Undo tree

This package turns the undo tangle emacs usually has into an undo tree 

(use-package undo-tree
:ensure t ;; install the package if it's not installed already
:config
(global-undo-tree-mode) ;; start undo-tree
)

3.2 EVIL

(use-package evil
:ensure t ;; install the package if it's not installed already
:init 
(setq evil-search-module 'evil-search)
(setq evil-want-integration t)
(setq evil-want-keybinding nil)
(setq evil-vsplit-window-right t) ; split to right
(setq evil-split-window-below t) ; split below
(setq evil-want-C-u-scroll t) ; scroll with ctrl-u
:config
(evil-mode 1)
)

3.3 EVIL-magit

Make evil work with magit 

(use-package evil-magit
:after evil
:ensure t ;; install the package if it's not installed already
)

3.4 EVIL collection

A collection of keybindings which cover the rest of the stuff evil doesn't 

(use-package evil-collection
:after evil ;; make sure it's started after evil
:ensure t ;; install the package if it's not installed already
:custom 
(evil-collection-setup-minibuffer t)
:config
(evil-collection-init)
)

3.5 Which-key

3.9 Helm

A popup finder 

(use-package helm
:ensure t ;; install the package if it's not installed already
:config 
(helm-mode 1)
(global-set-key (kbd "M-x") 'helm-M-x)
(global-set-key (kbd "C-x C-f") 'helm-find-files)
)

3.10 Projectile

3.11 Helm-projectile

For integration of helm and projectile 

(use-package helm-projectile
:ensure t ;; install the package if it's not installed already
:after (projectile helm)
:config 
(helm-projectile-on)
(setq projectile-completion-system 'helm)
)
